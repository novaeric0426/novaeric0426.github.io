FPS 개발일지 #04 — 플레이어 HP, 데미지, 죽음과 리스폰

체력 시스템, 데미지 처리, 사망/리스폰 로직 구현. 죽으면 2초 뒤 자동으로 되살아나는 리스폰 시스템.
2026-02-10 · 3 min · Alan

이번 글에서 다룰 것#

총을 쏘고 맞출 수 있게 됐으니, 이제 플레이어도 맞으면 죽어야 한다. HP 시스템, 데미지 처리, 사망 상태, 자동 리스폰을 구현한다.

Player 구조체 확장#

 1Player :: struct {
 2    // ... 기존 필드들 ...
 3
 4    // HP & Death
 5    hp:             f32,
 6    max_hp:         f32,
 7    is_alive:       bool,
 8    death_timer:    f32,
 9    spawn_position: Vec3,
10}

추가된 필드는 5개다:

  • hp / max_hp: 현재/최대 체력 (100)
  • is_alive: 생존 상태
  • death_timer: 리스폰까지 남은 시간
  • spawn_position: 처음 스폰 위치를 저장해두고, 리스폰할 때 사용

데미지 → 사망#

 1damage_player :: proc(player: ^Player, amount: f32) {
 2    if !player.is_alive do return
 3    player.hp -= amount
 4    if player.hp <= 0 {
 5        player.hp = 0
 6        kill_player(player)
 7    }
 8}
 9
10kill_player :: proc(player: ^Player) {
11    player.is_alive = false
12    player.death_timer = RESPAWN_DELAY  // 2.0초
13    player.velocity = {0, 0, 0}
14}

이미 죽어있으면 데미지를 무시한다. HP가 0 이하로 떨어지면 kill_player로 사망 처리하고, 속도를 0으로 만든다.

자동 리스폰#

 1RESPAWN_DELAY :: 2.0
 2
 3update_player_death :: proc(player: ^Player, dt: f32) {
 4    if player.is_alive do return
 5    player.death_timer -= dt
 6    if player.death_timer <= 0 {
 7        respawn_player(player)
 8    }
 9}
10
11respawn_player :: proc(player: ^Player) {
12    player.position = player.spawn_position
13    player.velocity = {0, 0, 0}
14    player.hp = player.max_hp
15    player.is_alive = true
16    player.death_timer = 0
17    player.yaw = 0
18    player.pitch = 0
19    player.move_state = .Walking
20    player.is_grounded = false
21    player.current_height = player.config.height_standing
22    sync_camera(player)
23}

사망 후 death_timer가 매 프레임 감소하다가 0이 되면 respawn_player가 호출된다. 리스폰은 모든 상태를 초기값으로 되돌린다 — 위치, 속도, 체력, 카메라 방향까지.

게임 루프에서의 처리 순서#

1// 매 프레임
2update_player_death(&game.player, dt)  // 죽은 상태면 타이머 감소
3
4if game.player.is_alive {
5    update_player(&game.player, &game.world, dt)
6    update_gun(&game.gun, &game.player, game.targets[:], &game.world, dt)
7}

사망 상태에서는 이동과 사격이 모두 무시된다. update_player_death만 돌면서 리스폰 타이머를 카운트다운한다.

사망 화면 UI#

1if !game.player.is_alive {
2    rl.DrawRectangle(0, 0, WINDOW_WIDTH, WINDOW_HEIGHT, rl.Color{200, 0, 0, 80})
3    rl.DrawText("YOU DIED", WINDOW_WIDTH/2 - 100, WINDOW_HEIGHT/2 - 30, 40, rl.RED)
4    timer_text := rl.TextFormat("Respawning in %.1f...", game.player.death_timer)
5    rl.DrawText(timer_text, WINDOW_WIDTH/2 - 90, WINDOW_HEIGHT/2 + 20, 20, rl.WHITE)
6}

죽으면 화면에 빨간 오버레이와 “YOU DIED” 텍스트, 리스폰 카운트다운을 표시한다.

HP 바 UI#

화면 하단에 체력 바를 그린다.

1hp_ratio := game.player.hp / game.player.max_hp
2bar_color: rl.Color
3if hp_ratio > 0.6       do bar_color = rl.GREEN
4else if hp_ratio > 0.3  do bar_color = rl.YELLOW
5else                     do bar_color = rl.RED

체력 비율에 따라 색이 변한다: 초록(60% 이상) → 노랑(30% 이상) → 빨강.

마치며#

이 글에서 구현한 것:

  • HP 시스템 (100 HP)
  • 데미지 처리 (중복 데미지 방지)
  • 사망 상태 (이동/사격 불가)
  • 2초 자동 리스폰 (전체 상태 초기화)
  • 사망 화면 UI + HP 바

다음 글에서는 히트마커와 데미지 넘버로 사격 피드백을 강화한다.
