Hugo + GitHub Pages로 개발 블로그 시작하기

Hugo와 PaperMod 테마를 이용해 GitHub Pages에 무료로 개발 블로그를 구축하는 방법을 소개합니다.
2026-03-05 · 1 min · Eric

왜 Hugo인가?#

Hugo는 Go로 작성된 정적 사이트 생성기(Static Site Generator)로, 빌드 속도가 압도적으로 빠릅니다.

  • 수천 개의 포스트도 수초 내 빌드
  • Markdown으로 콘텐츠 작성
  • GitHub Actions + GitHub Pages로 무료 배포

설치#

1# macOS
2brew install hugo
3
4# 버전 확인
5hugo version

새 프로젝트 생성#

1hugo new site my-blog
2cd my-blog
3git init

PaperMod 테마 추가#

1git submodule add https://github.com/adityatelange/hugo-PaperMod themes/PaperMod

로컬 서버 실행#

1hugo server -D

http://localhost:1313에서 확인할 수 있습니다.

💡 Tip: -D 플래그를 붙이면 draft: true인 포스트도 미리보기가 됩니다.

결론#

Hugo + PaperMod 조합은 개발 블로그에 최적입니다. 다음 포스트에서는 GitHub Actions를 이용한 자동 배포를 다뤄보겠습니다.
